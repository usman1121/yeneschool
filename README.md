# YeneSchool 🇪🇹
### The Definitive Integrated School Management System for Ethiopia

YeneSchool is a robust, all-in-one school enterprise resource planning (ERP) platform designed from the ground up to handle the unique local administrative, grading, and operational workflows of Ethiopian schools. 

Built for speed, accessibility, and precision, the platform unifies school ecosystems into a seamless digital infrastructure hosted securely via Vercel.

🌐 **Live Website:** [yeneschool.me](https://yeneschool.me)

---

## 🚀 Key Modules & Multi-Role Dashboards
YeneSchool provides distinct, customized workspaces for all school stakeholders:

*   **🔑 Admin (AD):** Complete oversight of institutional operations, system logs, and global configuration.
*   **📝 Registrar (RG):** Streamlined enrollment management, student profile tracking, and official record-keeping.
*   **🧑‍🏫 Teacher (TE):** Effortless grade entry, digital attendance tracking, and academic performance monitoring.
*   **📊 Student (ST):** Direct access to homework, schedules, performance records, and report cards.
*   **👪 Parent (PA):** Real-time monitoring of attendance, behavioral updates, and academic progress cards.
*   **💼 Finance (FI):** Automated fee collection, tuition invoicing, local payment tracking, and financial reporting.
*   **🛠️ IT Manager (IT):** System security configurations, permission management, and infrastructure stability.

---

## ✨ Features Tailored for the Ethiopian Market
- **Localized Workflows:** Built to match local grading frameworks, academic schedules, and administrative reporting requirements.
- **High Accessibility:** Lightweight, responsive design optimized to load instantly even on low-bandwidth networks.
- **Role-Based Security:** Strict data isolation and secure permission structures across all seven specialized user roles.

---

## 🛠️ Tech Stack & Architecture
- **Frontend & Deployment:** Modern HTML5, CSS3, JavaScript architecture deployed on **Vercel** for ultra-low latency and peak edge performance.
- **Backend & Database:** Scalable server architecture featuring a relational database schema optimized for multi-role concurrency, complex grading analytics, and financial ledger integrity.
- **Target Location:** Addis Ababa & across all regions of Ethiopia.
